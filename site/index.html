Introduction

Scalable is a fast, small, and feature-rich JavaScript library. It handles implementing a lot of the most popular web technologies with ease. You'll no longer deal with excessive testing and complex deployment distributions.

By leveraging the advanced, cloud based, platform of web 3.0, Scalable is able to quadruple site performance with limited hassle.
